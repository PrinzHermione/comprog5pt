Javascript

Javascript is a versatile and widely-used programming language that's primarily used for web development. it allows web developers to add interactivity,dynamic content, and functionality to websites

Objects

JS objects are key value pairs used to store and represent data. they are a fundamental data structure in Javascript,enabling you to group related information and access it by keys

Functions

JavaScript functions are blocks of reusable code that perform specific tasks or calculations when invoked. They allow you to encapsulate a set of instructions and execute them with a single function call.

Example

function addNumbers(a, b) { return a + b; } const result = addNumbers(5, 3); // The 'result' variable will now hold the value 8.

Conditionals

JavaScript conditionals are used to make decisions in your code. They allow you to execute different blocks of code based on whether a given condition is true or false. There are mainly two types of conditional statements in JavaScript: if statements and switch statements.

Example

if Statements: The if statement is the most basic conditional statement in JavaScript. It allows you to execute a block of code if a specified condition evaluates to true. let age = 18; if (age >= 18) { console.log("You are eligible to vote"); } else { console.log("You are not eligible to vote"); }

Loops

JavaScript loops are used to repeatedly execute a block of code as long as a specified condition is met. There are mainly four types of loops in JavaScript: for loops, while loops, do...while loops, and for...in loops. Each type of loop has its own use cases and can be helpful for different scenarios.

Example

let count = 0; while (count < 5) { console.log(count); count++; }
